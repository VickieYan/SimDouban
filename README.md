# SimDouban

https://wenjieyan.com/douban

一款基于 Vue.js 搭建的豆瓣仿站。

* 使用**Swiper.js**完成首页轮播图
* 使用**superagent**调用豆瓣接口数据，并用**superagent-jsonp**解决跨域问题
* 使用**InfiniteLoading**实现首页数据的懒加载
